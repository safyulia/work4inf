перемещаемся в какую нибудь папку, например /d/GitHub (сначала создаем ее)
 cd /d/GitHub
 ls (проверить где ты на линуксе)
 dir (проверить где ты на винде)
клонируем наш репозиторий
 git clone [ссылка на репозиторий с сайта github]
......
работаем в этой папке, оздаем разные файлы и тп. 
теперь, когда надо добавить файл
 git add (и написать чо за файл)/ git add --All
 git commit -m "коммент что мы сделали"
 git push. все (в первый раз попросит еще логин и пароль)
если надо наоборот - синхронизировать из облака 
 git pull
